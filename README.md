Sobre o Projeto Blue Origin

Este é um site institucional responsivo desenvolvido para a Blue Origin, focado na experiência de turismo espacial e carreiras no setor aeroespacial.


Desenvolvi este site, focando em:

-Interatividade com Mídia: Controle de execução de vídeo via código. 

-Dinâmica de Interface: Carrossel de imagens (slideshow) automático. 

-Experiência do Usuário: Uso de efeitos visuais modernos como glassmorphism e animações de entrada.

Tecnologias Utilizadas

-HTML5:Estrutura semântica e acessível.



-CSS3:Variáveis CSS e Flexbox. 



-JavaScript:Manipulação de eventos e DOM.



 Funcionalidades Implementadas



-Banner Dinâmico: Carrossel que alterna fotos do espaço automaticamente a cada 3 segundos. 



-Controle de Vídeo: Botão interativo para pausar e reproduzir o vídeo de fundo. 



-Menu Mobile: Menu estilo "hambúrguer" que se adapta para dispositivos móveis. 





-Validação de Inscrição: Formulário de rodapé que valida campos obrigatórios antes do envio. 



-Seção de Carreiras: Cards interativos com informações sobre diferentes engenharias e profissões.
